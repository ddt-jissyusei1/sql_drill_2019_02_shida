Q:貸し出し記録テーブル（rental）上に貸出日をキーにしたインデックスを作成してみましょう。
　以下の空欄を埋めて、SQL命令を完成させてください。

[ ① ]
 ind_rental
[ ② ]
 rental
 (
  [ ③ ]
 )
;


A:
CREATE INDEX		-- インデックス作成の命令句
 ind_rental			-- 作成するインデックスの名前の指定
ON					-- インデックス作成元を指定する句
 rental				-- 作成元となるテーブル名の指定
 (					-- 作成対象列の指定開始
  rental_date		-- 対象とする'貸出日'カラム名の指定
 )					-- 対象列指定の終了
;					-- SQL命令の終了
