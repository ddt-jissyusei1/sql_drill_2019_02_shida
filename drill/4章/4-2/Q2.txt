Q:注文書テーブル（order_main）上に発注日、納品日をキーにした複合インデックスを作成してみましょう。
　以下の空欄を埋めて、SQL命令を完成させてください。

[ ① ]
 ind_order
ON
 [ ② ]
 (
  [ ③ ]
 )
;


A:
CREATE INDEX		-- インデックス作成の命令句
 ind_order			-- 作成するインデックスの名前の指定
ON					-- 作成元の指定句
 order_main			-- 作成元となるテーブル名の指定
 (					-- 作成対象となる列指定の開始
  order_date,		-- 対象とする'発注日'カラム名の指定
  delivery_date		-- 対象とする'納品日'カラム名の指定
 )					-- 対象列指定の終了
;					-- SQL命令の終了
